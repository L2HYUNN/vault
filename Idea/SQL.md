## 데이터베이스와 SQL
**데이터베이스(Database):** 데이터의 집합
**DBMS(Database Management System):** 데이터베이스를 관리하고 운영하는 소프트웨어
	- 대용량 데이터를 관리해야한다.
	- 여러 사용자와 공유할 수 있어야한다.
**SQL(Structured Query Language):** DBMS에 데이터를 구축, 관리하고 활용하기 위해 사용되는 언어
**RDBMS(Relational DBMS):** 테이블이라는 최소 단위로 구성된 데이터베이스
